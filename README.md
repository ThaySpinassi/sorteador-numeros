# 🔢 Sorteador de Números

## 📄 Sobre
<p>Este projeto é um sorteador de números aleatórios simples. Ele permite ao usuário definir um intervalo de números e a quantidade de números a serem sorteados. </p>
<p>Projeto utilizado no curso de <b>Lógica de Programação</b> da Alura.</p>

## ⚙️ Funcionalidades
* **Sorteio de números aleatórios:** Gera números aleatórios dentro de um intervalo especificado.
* **Evita repetições:** Garante que os números sorteados sejam únicos.
* **Interface simples:** Possui uma interface intuitiva para o usuário definir os parâmetros do sorteio.
